# How to use GIT ?

## How to create a local repository ?

1. mkdir ~/Nameofrepo
2. cd ~/Nameofrepo
3. git init

## Make your first commit to Github

1. touch README.md
2. git status
3. git add README.md
4. git commit -m "Add README.md"
5. git remote add origin git@github.com:rohmerpauline/Nameofrepo
6. git push
